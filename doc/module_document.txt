### class noise_rain_haze_degradation

module_name: noise_rain_haze_degradation
location: /home/huhao/adv_ir/source/degradation.py
class_name: noise_rain_haze_degradation (nn.Module)

purpose:
- unified controller for noise/rain/haze degradations with per-reset random state

constructor:
- noise_config, rain_config, haze_config: each value supports fixed / pair-range / discrete candidates
- enable_noise_prob, enable_rain_prob, enable_haze_prob

reset_behavior:
- signature: reset_parameters(enable_noise=None, enable_rain=None, enable_haze=None)
- samples enable flags (unless explicitly overridden)
- samples current params for all three submodules
- applies sampled params to submodules and resets submodule states
- execution order rule:
	- rain/haze random order when both enabled
	- noise always last when enabled

forward_method:
- signature: forward(image, distance_map=None, rain_degraded_list=None, rain_topk=None) -> Tensor
- returns only degraded image tensor
- haze requires distance_map
- rain requires rain_degraded_list

regularization:
- get_regularization_loss
- get_regularization_items with keys: loss_noise, loss_rain, loss_haze, loss_total

state:
- get_current_state returns enable flags, current order, and sampled params
