### function random_degradation_configs_from_image

module_name: random_degradation_configs_from_image
location: /home/huhao/adv_ir/source/degradation.py
class_name: function

purpose:
- given one image tensor, sample a valid triplet of Noise/Rain/Haze dataclass configs

input:
- image shape: (B, C, H, W) or (C, H, W)

sampling_rules:
- noise_strength in {15, 25, 50} / 255
- rain router num_maps in {4, 8, 16}
- haze airlight base in [0.85, 1.0], per-channel jitter +-0.02, channel clamp to [0.85, 1.0]
- haze beta_mean in [1.0, 1.3]
- haze beta_std in [0.05, 0.2]

return:
- (NoiseDegradationConfig, RainDegradationConfig, HazeDegradationConfig)

### class noise_rain_haze_degradation

module_name: noise_rain_haze_degradation
location: /home/huhao/adv_ir/source/degradation.py
class_name: noise_rain_haze_degradation (nn.Module)

purpose:
- unified controller for noise/rain/haze degradations with explicit enable flags

constructor:
- noise_config: NoiseDegradationConfig
- rain_config: RainDegradationConfig
- haze_config: HazeDegradationConfig
- enable_noise, enable_rain, enable_haze: bool

reset_behavior:
- signature: reset_parameters()
- resets all three submodules
- rebuilds execution order
- execution order rule:
	- rain/haze random order when both enabled
	- noise always last when enabled

forward_method:
- signature: forward(image, distance_map=None, rain_degraded_list=None, rain_topk=None) -> Tensor
- returns only degraded image tensor
- haze requires distance_map
- rain_degraded_list is optional (None triggers auto rain candidate generation in rain module)

regularization:
- get_regularization_loss
- get_regularization_items with keys: loss_noise, loss_rain, loss_haze, loss_total

state:
- get_current_state returns enable flags, current order, and config dicts (noise_config/rain_config/haze_config)
