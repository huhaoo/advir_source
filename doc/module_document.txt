- /home/huhao/adv_ir/tmp_demo/haze_dataset_builder/summary.json
notes:
- saved density images include inline value labels: min/max/mean.
- density profile is adaptive and monotonic outer->inner.
- random per-call ranges:
  - edge inset: [0.02, 0.14]
  - q_temp: [0.20, 0.80]
  - notch strength: [0.10, 0.32]
  - amp_max: [0.30, 0.50]
- adv helper now supports fixed output mean:
  - adv(grad, lam1, lam2, size, fixed_output_mean=0.2)
  - output mean is enforced by additive projection so matrix mean equals fixed_output_mean
  - optimization style aligned to run_single_image_adversarial_degradation_search:
    - steps1 is fixed to 1
    - if grad is tuple/list, uses the first gradient as fixed_grad
    - no softplus re-parameterization
    - fixed_grad is centered and rescaled to target std from /home/huhao/adv_ir/source/doc/static.txt
      (target std = 9.64294260850096e-05)
    - fixed-mean constraint is applied during inner optimization and final output
