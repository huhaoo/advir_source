### degradation
module_name: degradation
location: /home/huhao/adv_ir/source/degradation.py
kind: module
summary: haze-only degradation operator + differentiable motion-blur operator + random haze config sampler; supports parameterized interpolation override for control maps.
run: imported by training/attack scripts
outputs: runtime tensors and regularization terms
notes:
- interpolation parameters are explicit function arguments:
  - random_haze_degradation_config_from_image(image, interp_mode, gaussian_radius, gaussian_sigma, gaussian_extra_cells)
  - random_degradation_configs_from_image(...) is kept as a haze-only alias for compatibility
  - default values keep current behavior: bicubic + align_corners=False path
- differentiable motion blur module:
  - motion_blur_degradation(MotionBlurDegradationConfig(...))
  - Haze-style adversarial controller with internal trainable dx map from two control maps
  - input: image [B,C,H,W], optional dx [B,2,H,W] override
  - output pixel equals centered line average from p-dx(p)/2 to p+dx(p)/2 via K-point differentiable sampling
  - defaults: num_steps=16, batchify_steps=true, dmax=None, dlambda=0.0
  - dmax semantics:
    - dmax is None: no clipping
    - dmax >= 0: clip with this absolute pixel value
    - dmax < 0: effective_dmax = (-dmax) * diagonal_length
  - regularization:
    - map regularization from both control maps
    - dx magnitude regularization weighted by dlambda
  - projection:
    - project_trainable_parameters_() projects gaussian offsets and clamps low-res dx when dmax is set
